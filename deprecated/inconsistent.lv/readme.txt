commandline arguments:

-map   specifies the map file
-v   specifies the algorithm to run, if not specified, all algorithms are run
-delta specifies the parameter for approximation alg
-N  specifies the parameter for the figure, if -fig is set
-fig  specifies which figure will be used, 1 for Materlli, 2 for Mero fig1, 3 for Mero fig2

Most parameters have default values, so it's ok to not set them.
See mymain.cpp for default values.
